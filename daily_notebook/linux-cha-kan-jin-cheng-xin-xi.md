# linux查看进程信息

* 查看端口监听

  `netstat -tunlp`

![](../.gitbook/assets/20190414-150435-de-ping-mu-jie-tu.png)

* 查看进程详细信息

  根据进程ID（PID）查看进行的详细信息，也可以根据**程序的名称**来查找程序

  `ps -ef | grep 进程ID/名称`
